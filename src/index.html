What is flx-helper ?

flx-helper is a css3 flexbox lightweight util that provides some helper classes in order to speed up your layouts using flex.

How does it work ?

  1. Just add the flx class into the html element you want to be flex
  2. You can change the flexbox behavior just adding modifiers classes
  3. You can also use a single class preset in order to reduce the usage of multiple modifiers classes

Modifiers list:

  • flx--inline Set inline flex
  • flx--row Set flex-direction row
  • flx--row-r Set flex-direction row-reverse
  • flx--col Set flex-direction col
  • flx--col-r Set flex-direction col-reverse
  • flx--wrap Set flex wrap
  • flx--wrap-r Set flex wrap reverse
  • flx--spc-around Set flex space-around
  • flx--spc-btw Set flex space-between
  • flx--stretch Set flex align-items to stretch
  • flx--bs-line Set flex align-items to baseline
  • flx--gr-1 Set flex grow to 1
  • flx--gr-0 Set flex grow to 0
  • flx--sh-1 Set flex shrink to 1
  • flx--sh-0 Set flex shrink to 0
  • flx--bs-a Set flex basis to auto
  • flx--bs-0 Set flex basis to 0

Single class preset list:

Single class presets allows you to use only one class in order to achieve an specific and frequently used layout.
